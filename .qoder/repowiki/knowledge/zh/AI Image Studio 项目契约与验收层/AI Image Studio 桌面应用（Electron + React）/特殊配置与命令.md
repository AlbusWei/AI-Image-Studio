开发环境 `npm run dev` 通过 concurrently 并行启动 Vite（127.0.0.1:5173, --strictPort）和 Electron，并用 wait-on 等待 Vite 就绪后再加载 `http://127.0.0.1:5173`；生产构建 `npm run electron:build` 先执行 `vite build` 再调用 `electron-builder --win --config electron-builder.yml`。环境变量集中定义在 `.env`（`VITE_QWEN_API_KEY`、`VITE_EVOLINK_API_KEY`、`VITE_OSS_*`、`VITE_EXPANSION_LLM_*` 等），由 main 进程 api-server 与 renderer 侧 useSettingsStore 共同读取。
