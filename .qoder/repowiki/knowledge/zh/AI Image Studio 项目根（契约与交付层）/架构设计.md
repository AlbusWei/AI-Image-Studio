本模块不含业务源码，仅承载面向 `app/` 的单向契约层：
- `docs/PRD-AI-Image-Studio.md` 与 `docs/CONTEXT.md` 定义产品需求与领域术语，`docs/adr/` 记录架构决策，构成 `app/` 实现的需求基线。
- `ai-image-studio-summary.canvas.tsx` + `*.canvas.status.json` 基于 `qoder/canvas` 组件库渲染交互式完成度报告，将 P0/P1 功能、里程碑、技术栈与 Bug 修复清单以可执行页面形式呈现，作为交付物入口。
- 大量 `e2e-*.png` 截图按测试阶段命名（initial/load/generate/fail/pass/verify），覆盖 Qwen/GPT/Nano Banana 三模型生图、Workbench 创作流、Gallery 搜索筛选、知识库案例包、设置页等关键路径，作为视觉回归证据与 PR 审查附件。
- `image studio.code-workspace` 提供 VS Code 工作区配置，统一 IDE 打开方式。
依赖方向严格单向向外：仅引用自身静态资源与 `docs/`，不反向依赖 `app/` 源码。
