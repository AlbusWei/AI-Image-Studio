## QoderWork Skills

This project ships with QoderWork agent skills for AI-assisted workflows (e.g., batch image generation with parallel dispatch and VLM-based QA). To install:

**Windows:**
```bat
scripts\install-skills.bat
```

**macOS / Linux:**
```bash
bash scripts/install-skills.sh
```

See [docs/QODERWORK-SKILLS.md](./docs/QODERWORK-SKILLS.md) for full details, manual install steps, and sync instructions.
