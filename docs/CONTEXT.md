# AI Image Studio — 领域模型（统一术语表）

本文档定义产品设计中的统一术语，所有产品文档、交互设计、开发实现必须使用此处定义的规范名称。

---

**Studio**
AI 图片生成工作站的总称，即本产品本身。
_Avoid_：工具、平台、应用（过于泛化）

**工作区（Workbench）**
Studio 的中央创作区域，包含提示词编辑器、参考图区、参数面板、结果画板。用户在此完成从输入 prompt 到审视生成结果的全部创作活动。
_Avoid_：生成页面、创作页、编辑器

**结果画板（Result Board）**
工作区内的子区域，展示当前批次的生成结果。支持网格展示、大图预览、批次切换。
_Avoid_：结果区域、图片展示区

**生成任务（Generation Task）**
一次提交到模型 API 的生图请求，是后台调度的最小单位。具有状态生命周期：排队中 → 生成中 → 完成 / 失败。
_Avoid_：生成请求、生成作业

**批次（Batch）**
同一次用户操作产生的一组生成任务的集合及其结果。批次是结果展示和审视的基本单位。
_Avoid_：一组图、一批、这轮

**提示词（Prompt）**
用户输入的自然语言描述，用于指导模型生成图片。
_Avoid_：咒语、描述语

**原始提示词（Original Prompt）**
用户直接输入的、未经任何扩写处理的提示词文本。
_Avoid_：原始描述、用户输入

**扩写提示词（Expanded Prompt）**
经 LLM 扩写助手处理后，或经模型端 prompt_extend 处理后，最终发送给生图模型的完整提示词。
_Avoid_：改写 prompt、增强 prompt、最终 prompt

**提示词扩写助手（Prompt Expansion Assistant）**
工作区内的主动扩写功能，调用 LLM + RAG 案例库，将原始提示词扩写为一个或多个候选版本供用户选择。
_Avoid_：AI 改写、智能提示、prompt 增强

**参考图（Reference Image）**
用户提供的、用于指导图片生成方向的输入图片。不同模型支持的数量不同（Qwen Image 3: 1-3 张，GPT-image-2: 1-16 张，Nano Banana 2: 1 张）。
_Avoid_：输入图、底图、素材图

**遮罩（Mask）**
一张与参考图尺寸一致的 PNG 图片，包含 alpha 通道。透明区域表示需要重新生成的区域，不透明区域表示保持不变的区域。仅 GPT-image-2 支持。
_Avoid_：蒙版、选区图

**局部重绘（Inpainting）**
基于参考图 + 遮罩，仅重新生成遮罩透明区域的内容，其余区域保持不变。仅 GPT-image-2 支持。
_Avoid_：局部修改、部分重绘、区域编辑

**模型（Model）**
可调用的 AI 图片生成服务。当前支持三种：Qwen Image 3、GPT-image-2、Nano Banana 2。
_Avoid_：引擎、AI、算法

**模型能力矩阵（Model Capability Matrix）**
描述各模型在参考图数量、尺寸选项、分辨率、质量档位、mask、一次出图数等维度上差异的对照表。产品 UI 据此控制参数的可用/不可用状态。
_Avoid_：功能对比表、参数表

**画廊（Gallery）**
Studio 的内容浏览模块，展示所有已生成的图片、参考图和导入的外部图片。支持时间线浏览、智能分组、多维检索。
_Avoid_：图库、相册、图片列表

**批次会话（Batch Session）**
Gallery 时间线中的智能分组单位——将同一次创作操作产生的所有批次归为一组。
_Avoid_：会话、分组、创作组

**文件夹（Folder）**
Gallery 中的多级目录结构，是图片的物理归属。每张图片有且仅有一个所属文件夹。支持无限级嵌套。
_Avoid_：目录、分类、分组

**虚拟视图（Virtual View）**
不改变图片物理归属、基于条件动态聚合的展示视图。系统内置的虚拟视图包括：全部图片、最近生成、收藏。
_Avoid_：智能相册、动态分类

**收藏（Favorite）**
图片的全局标记（星标），与文件夹归属无关。被收藏的图片在"收藏"虚拟视图中集中展示。
_Avoid_：点赞、标星、喜欢

**知识库（Knowledge Base）**
从收藏中精选的高质量案例集合，作为提示词扩写 RAG 检索的素材源。三层漏斗的最上层：全部图片 → 收藏 → 知识库。
_Avoid_：案例库、素材库、成功库

**案例包（Case Package）**
知识库中的一条完整记录，包含：生成图片 + 原始提示词 + 扩写提示词 + 参考图 + 模型名 + 全部参数 + 用户标注（可选）。是 RAG 检索和知识复用的最小单位。
_Avoid_：案例、记录、条目

**用户标注（User Annotation）**
附加在案例包上的文本描述，说明该案例的成功经验、适用场景或可复用技巧。可由用户手写，也可由多模态 LLM 辅助生成草稿。
_Avoid_：备注、评论、标签

**热区（Hot Zone）**
本地存储区域，保留原图以确保零延迟浏览。容量可配置（默认基于可用磁盘空间智能推荐，用户视角约 100GB）。
_Avoid_：本地缓存、本地存储

**冷区（Cold Zone）**
云端存储区域，仅保留原图。本地只保留缩略图和元数据。查看原图时需从云端拉取。
_Avoid_：归档、云存储、远端

**任务指示器（Task Indicator）**
页面角落的浮动图标 + 数字角标，显示当前后台任务数量。悬浮显示摘要，点击展开任务面板。
_Avoid_：任务图标、状态图标

**任务面板（Task Panel）**
点击任务指示器后展开的面板，展示所有进行中、排队中、已完成、失败的生成任务及其详情。
_Avoid_：任务列表、任务中心、下载管理器

**Toast 通知**
任务完成时在页面角落弹出的轻量通知，3 秒后自动消失。包含"查看"按钮可跳转结果。
_Avoid_：弹窗、通知框、提醒

**成功提示词（Success Prompt）**
产生被用户收藏/标记为优秀的生成结果时使用的最终提示词。自动入库成为 RAG 检索的素材。
_Avoid_：好的 prompt、优质提示词

**种子值（Seed）**
控制生成随机性的数值参数。相同 prompt + 相同参数 + 相同 seed = 确定性生成结果（取决于模型是否支持）。用于复现和对比。
_Avoid_：随机种子、噪声种子

**批量生成（Batch Generation）**
覆盖三种模式的统称：多批次（同一 prompt 重复生成）、多变体（参数排列组合）、Prompt 队列（多 prompt 依次执行）。
_Avoid_：批量出图、批量任务、批量渲染

**设置页（Settings Page）**
集中管理所有配置的中心页面，包含模型 API 配置、存储策略、扩写 LLM 配置、通用偏好等分 Tab 组织。
_Avoid_：配置页、偏好设置、选项

**导入 / 导出（Import / Export）**
数据进出 Studio 的能力。支持图片、元数据、知识库、文件夹结构、配置的完整迁移。
_Avoid_：上传下载、备份恢复

**快捷键（Keyboard Shortcut）**
绑定到键盘按键或组合键的操作指令，按焦点区域上下文感知生效。分为全局快捷键、工作区快捷键、Gallery 快捷键、Lightbox 快捷键、Mask 编辑器快捷键五个层级。
_Avoid_：热键、快捷键（不够精确，部分快捷键不含 Ctrl/Alt 修饰键）

**快捷键速查浮层（Shortcut Cheat Sheet）**
按 `?` 键调出的浮层面板，按场景分组展示所有可用快捷键，当前焦点所在场景的组高亮显示。
_Avoid_：快捷键帮助、快捷键面板

**焦点区域（Focus Zone）**
当前接收键盘事件的 UI 区域。决定哪些快捷键生效。优先级从高到低：Mask 编辑器 > Lightbox > 工作区输入框 > Gallery 网格 > 全局。
_Avoid_：焦点上下文、活动区域
